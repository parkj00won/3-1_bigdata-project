|---|--------|------------|------|

### 2-2. 결측치 현황

| 컬럼 | 결측치 수 | 비율 |
|------|----------|------|
| 성별 | 185,977 | 31.0% |
| 기타 컬럼 | 0 | 0% |

**숫자 변환 불가능한 값 확인 (운동량, 탄소량):**

| 컬럼 | 비정상 값 | 건수 | 원인 |
|------|----------|------|------|
| 운동량 | `\N` | 475건 | DB에서 NULL을 `\N` 문자열로 내보내어 object 타입이 됨 |
| 탄소량 | `\N` | 475건 | DB에서 NULL을 `\N` 문자열로 내보내어 object 타입이 됨 |

> `운동량`과 `탄소량` 컬럼의 데이터 타입이 실수(float)가 아닌 object인 이유:
> 원본 CSV에 MySQL 등 DB의 NULL 표현인 `\N` 문자열이 475건 섞여 있어,
> Pandas가 컬럼 전체를 문자열(object)로 인식했기 때문이다.
> 전처리 단계에서 `pd.to_numeric(errors="coerce")`로 변환하면 `\N`은 NaN으로 처리된다.

**결측치 히트맵:**

![결측치 히트맵](bike_analysis/bike_01_missing_heatmap.png)

**해석:**
- 성별 컬럼에만 결측치가 집중적으로 발생 (31%)
- 이는 **일일권(비회원)** 이용자의 경우 회원 정보가 없어 성별이 기록되지 않기 때문
- 비회원 이용 데이터 185,977건은 성별 분석 시 별도로 처리 필요
- 운동량/탄소량의 `\N` 값 475건은 히트맵에서 탄소량 컬럼 부근에 미세하게 관찰됨

- 최댓값이 극단적 (이용건수 최대 1,875건, 이동거리 최대 약 9,295km) → 이상치 존재 가능

- 4월 이후 증가율이 점차 완만해지는 추세

### 5-2. 연령대별 이용건수 (월별 비교)

![연령대별 이용건수](bike_analysis/bike_03_age_usage.png)

**해석:**

- **20대와 30대**가 모든 월에서 가장 높은 이용건수를 기록 (6월 기준 각 약 100만 건 이상)
- 40대, 50대도 상당한 이용량을 보이며, 특히 봄~여름 시즌에 증가폭이 큼
- 70대 이상은 전 연령대 중 가장 낮지만, 월별 증가 추세는 동일
- **~10대**는 절대량은 적지만 봄~여름에 증가폭이 뚜렷 → 학생층 레저 이용 추정

![성별 이용건수 비교](bike_analysis/bike_04_gender_pie.png)

**해석:**
- **남성 64.5% (8,488,405건)** vs **여성 35.5% (4,663,053건)**
- 성별이 확인된 이용 건수 기준으로 남성이 약 1.8배 더 많이 이용

### 5-4. 수치형 변수 분포

![수치형 변수 분포](bike_analysis/bike_05_distributions.png)

**해석:**
- 세 변수(이용건수, 이동거리, 이용시간) 모두 **강한 양의 왜도(right-skewed)** 분포
- 대부분의 데이터가 낮은 값에 집중되고, 소수의 높은 값이 긴 꼬리를 형성
- **이용건수**: 대부분 50건 이하에 집중, 소수의 인기 대여소에서 100건 이상 발생
- **이동거리**: 대부분 100km 이하, 일부 극단적으로 긴 이동거리 존재
- **이용시간**: 대부분 1,000분 이하에 집중

> 상위 1%를 제외하고도 긴 꼬리가 관찰되어, 이상치 탐지 및 처리가 후속 분석에서 중요

### 5-5. 월별 이동거리 분포 (박스플롯)

![월별 이동거리 박스플롯](bike_analysis/bike_06_boxplot_distance.png)

**해석:**
- 모든 월에서 **이상치(outlier)** 가 다수 존재 (상위 1% 제외 후에도)
- 봄~여름(3~6월)으로 갈수록 **IQR(사분위범위)이 넓어짐** → 이용 패턴이 다양해짐
- 중앙값(박스 내부 선)은 1~2월에 비해 3~6월이 높아져 전반적 이동거리 증가
- 1~2월: 상자가 좁고 중앙값이 낮음 → 짧은 거리 위주의 통근/통학 이용 추정
- 4~6월: 상자가 넓고 위스커가 김 → 레저/장거리 이용 증가

## 6. 상관관계 분석

### 6-1. 상관계수 행렬

| | 이용건수 | 운동량 | 탄소량 | 이동거리(M) | 이용시간(분) |
|---|---------|-------|-------|-----------|------------|
| **이용건수** | 1.00 | 0.85 | 0.85 | 0.85 | 0.88 |
| **운동량** | 0.85 | 1.00 | 0.99 | 0.99 | 0.95 |
| **탄소량** | 0.85 | 0.99 | 1.00 | 1.00 | 0.96 |
| **이동거리(M)** | 0.85 | 0.99 | 1.00 | 1.00 | 0.96 |
| **이용시간(분)** | 0.88 | 0.95 | 0.96 | 0.96 | 1.00 |

### 6-2. 주요 발견

- **모든 수치형 변수 간에 강한 양의 상관관계** (r >= 0.85)
- **탄소량 ↔ 이동거리: r = 1.00** → 탄소 절감량은 이동거리로부터 직접 계산된 파생 변수
- **운동량 ↔ 이동거리: r = 0.99** → 운동량도 이동거리 기반 산출 추정
- **이용건수 ↔ 이용시간: r = 0.88** → 이용 횟수가 많을수록 총 이용시간도 비례 증가

### 6-3. 검토 의견

| 검토 항목 | 내용 |
|----------|------|
| **다중공선성** | 운동량, 탄소량, 이동거리 세 변수가 r >= 0.99로 거의 동일한 정보를 담고 있음. 머신러닝 모델링 시 이 중 하나만 선택하거나 차원 축소 필요 |
| **파생 변수 확인** | 탄소량과 이동거리의 r=1.00은 `탄소량 = 이동거리 × 상수`로 계산되었음을 시사. 분석 시 중복 변수 제거 권장 |
| **이용건수의 독립성** | 이용건수는 다른 변수들과 0.85~0.88의 상관관계로, 가장 독립적인 변수. 이용 패턴 분석의 주요 타겟 변수로 적합 |

## 7. 인사이트 요약

### 핵심 발견사항

1. **계절 효과**: 겨울(1~2월) 대비 여름(6월)에 이용량이 약 2.4배 증가. 기온과 자전거 이용량 사이의 강한 양의 관계가 확인됨

2. **이용자 프로필**: 20~30대 남성이 핵심 이용층. 남성 이용 비율(64.5%)이 여성(35.5%)의 약 1.8배

3. **이용권 구조**: 정기권(55.2%)이 과반으로, 출퇴근/통학 등 일상적 이동수단으로 정착. 일일권(41.8%)도 높은 비율

4. **데이터 분포 특성**: 모든 수치형 변수가 오른쪽 꼬리 분포를 보이며, 소수의 인기 대여소에서 대량 이용이 발생하는 **롱테일(long-tail)** 구조

5. **결측치 패턴**: 성별 결측(31%)은 비회원 이용자에서 발생하며, 데이터 품질 문제가 아닌 수집 구조에 기인

6. **변수 중복성**: 운동량, 탄소량, 이동거리는 사실상 동일한 정보를 담고 있어 분석 시 하나만 사용해도 충분

### 추가 분석 방향 제안

| 방향 | 설명 |
|------|------|
| **대여소별 인기 순위** | 이용건수 기준 상위 대여소 분석 → 자전거 배치 최적화 |
| **시계열 세분화** | 월별 → 일별/요일별 분석으로 평일/주말 패턴 비교 |
| **비회원 분석** | 성별 결측(비회원) 데이터를 별도 그룹으로 분석 |
| **이상치 처리** | IQR 기반 이상치 탐지 후 제거/변환하여 정제된 통계 산출 |
| **외부 데이터 연계** | 기상 데이터(기온, 강수량)와 결합하여 날씨-이용량 관계 분석 |
| **클러스터링** | 대여소별 이용 패턴 군집분석으로 유형 분류 |
