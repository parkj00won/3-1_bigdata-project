> **주의**: `transformers` 5.x는 `tensorflow_text` 백엔드 오류, `torch` 2.11+는 `meta tensors` 오류가 발생합니다. 반드시 위 버전을 지정하세요.

### HuggingFace 계정 및 CLI 로그인

#### 계정 가입

1. https://huggingface.co 접속 → 우측 상단 **Sign Up** 클릭
2. 이메일, 사용자명, 비밀번호 입력 후 가입 완료
3. 이메일 인증 링크 클릭

> 계정이 있으면 모델 다운로드 속도 제한이 완화되고, 비공개 모델 접근 및 Spaces 배포가 가능합니다.

#### Access Token 발급

1. https://huggingface.co/settings/tokens 접속 (또는 프로필 → Access Tokens)
2. **New token** 클릭
3. 이름 입력 (예: `my-laptop`), 권한은 `Read` 선택 → **Generate**
4. 생성된 토큰(`hf_...`)을 복사하여 안전한 곳에 보관

> ⚠️ 토큰은 비밀번호와 같습니다. 코드에 직접 작성하지 말고, 환경 변수나 `secrets.toml`에 저장하세요.

#### CLI 로그인

**중요**: 반드시 **venv 가상 환경이 활성화된 터미널**에서 실행하세요!

```bash
# 1. 터미널에서 로그인 (Access Token 입력)
hf auth login
# → "Enter your token:" 프롬프트가 나오면 위에서 복사한 토큰 붙여넣기
# → 입력해도 화면에 표시되지 않으니, 붙여넣기 후 엔터를 누르고 출력 메시지를 확인
# → "Add token as git credential? [y/N]:" y 입력

# 2. 로그인 확인
hf auth whoami
# → 사용자명이 표시되면 성공
```

> 로그인 정보는 `~/.cache/huggingface/token`에 저장되어 이후 자동으로 인증됩니다.

### GPU 환경 (CUDA)

#### CUDA 버전 확인 방법

```bash
# 방법 1: nvidia-smi (드라이버가 지원하는 최대 CUDA 버전)
nvidia-smi
# 출력 우측 상단의 "CUDA Version: 12.x" 확인

# 방법 2: nvcc (실제 설치된 CUDA Toolkit 버전)
nvcc --version
# "release 12.1" 등으로 표시됨

# 방법 3: Python에서 확인
python -c "import torch; print('CUDA 사용 가능:', torch.cuda.is_available()); print('CUDA 버전:', torch.version.cuda)"
```

> **nvidia-smi vs nvcc**: `nvidia-smi`는 GPU 드라이버가 지원하는 최대 CUDA 버전, `nvcc`는 실제 설치된 CUDA Toolkit 버전입니다. torch 설치 시에는 `nvcc` 버전에 맞추세요.

#### CUDA 버전별 설치

```bash
# CUDA 12.1 이상
pip install torch==2.6.0 torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# CUDA 11.8
pip install torch==2.6.0 torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```
